# Proyecto_Modelacion_Matematica_Sistemas_Biologicos

Proyecto en R para la clase Modelacion matematica de sistemas biologicos.

## Objetivo

Generar un modelo matematico que describa la via de TGF beta y sus ciclos de retroalimentacion.
